Forge

One system for planning, execution, memory, and review.

Forge keeps goals, projects, tasks, habits, task runs, notes, wiki pages, sleep records, workout records, preferences, and Psyche records in one local system. OpenClaw and Hermes can read that data, update it, and hand off to the Forge UI when the UI is the better surface.

What You Get

What Forge includes.

This is the short product map. Use the features page for the longer reference.

Planning

Goals, projects, tasks, habits, strategies, notes, and calendar records stay linked in one graph.

Execution

Task runs track real work. Work adjustments fix logged minutes without inventing a fake run.

Wiki memory

The wiki stores durable pages with backlinks, search, health checks, and background ingest jobs.

Health

Sleep and sports are first-class record types with notes, tags, and Forge links.

Preferences

Preferences includes comparison rounds, direct signals, and editable concept libraries.

Psyche

Values, beliefs, patterns, modes, and trigger reports stay in the same system as work and review.

OpenClaw

Hermes

Native Python plugin. Best if Hermes is your main agent runtime.

Standalone

Getting Started

Install Forge in OpenClaw.

Start with the normal install path. If OpenClaw 2026.4.x blocks it, use the temporary bypass directly below.

Setup

1. Install

Install the published OpenClaw plugin and enable it.

2. Allow

Add the plugin to plugins.allow if it is not already present.

3. Restart and check

Restart the gateway, run the health check, then open the UI.

Important for OpenClaw

openclaw plugins enable forge-openclaw-plugin is not always enough by itself. If forge-openclaw-plugin is missing from plugins.allow, OpenClaw can still refuse to load it.

What the check confirms

openclaw forge health confirms the plugin loaded, the Forge runtime is reachable, and the local API is answering.

Default local UI

The common local UI address is http://127.0.0.1:4317/forge/.

Default data path

OpenClaw stores Forge under ~/.openclaw/extensions/forge-openclaw-plugin/data/forge.sqlite unless you set a custom dataRoot.
